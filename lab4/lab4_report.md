# Лабораторная работа №4 "Разработка инфраструктуры MVP AI приложения"

## 1. Цели и задачи работы
**Цель:** создать прототип AI-приложения с базовой функциональностью.
**Задачи:**
1. Предоставить исходные данные о приложении
2. Выбрать необходимые ресурсы с обоснованием
3. Продумать начальное состояние приложения и составить схему его инфраструктуры в draw.io
5. Рассчитать расходы на содержание начального состояния
6. Продумать состояние приложения для тестирования партнерами и составить схему его инфраструктуры в draw.io
7. Рассчитать расходы на содержание для тестового состояния приложения
8. Продумать продовое решение приложения и составить схему его инфраструктуры в draw.io
9. Рассчитать расходы на содержание для продового решения
10. Рассчитать предполагаемую нагрузку для трех решений приложения
11. Выбрать оптимальное решение изходя из предполагаемых нагрузок
12. Подвести итоги
13. Составить отчет по лабораторной работе
